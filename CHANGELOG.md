# Changelog

## [v1.2.0] - 2026-01-20

### Added
- Directional swipe support (left/right)
- Smooth thumb animation on mode change
- Programmatic mode switching API
- Icon switching based on active mode
- Accessibility announcements for mode changes

### Changed
- Simplified swipe logic (removed free-drag behavior)
- Cleaner and more stable touch handling
- Reduced public API surface for stability

### Removed
- Free drag thumb movement
- Overlapping gesture logic
- Unstable layout-dependent calculations

### Fixed
- Thumb sticking to edges
- Inconsistent padding behavior
- Touch interception issues
- Accessibility crash on disabled services
